# Demo Page 素材指南

將圖片放到對應資料夾，命名如下。放好後告訴 Claude 即可自動上傳並更新 demo pages。

---

## demo-app-landing/ (AppAI App Landing Page)

| 檔名 | 用途 | 建議尺寸 | 說明 |
|------|------|----------|------|
| `hero-bg.png` | Hero 背景圖 | 1920x1080 | 深色漸層或科技感背景 |
| `screenshot-1.png` | 產品截圖 1 | 1280x720 或手機尺寸 | 例：AppAI 首頁畫面 |
| `screenshot-2.png` | 產品截圖 2 | 同上 | 例：Dashboard 畫面 |
| `screenshot-3.png` | 產品截圖 3 | 同上 | 例：Hosted page 效果 |
